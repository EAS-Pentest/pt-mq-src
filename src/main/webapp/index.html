Sample Liberty Application Connects to IBM MQ on Cloud

Run the sample application via Curl command with the deployed application URL from your EASe Java instance

curl -X POST -d 'msg=testing messages with IBM MQ on Cloud' https:///libertymq/api/enqueue

for example: curl -X POST -d 'msg=testing messages with IBM MQ on Cloud' https://mei-easej-mq-academic-way-8529.staging.us-east.c2.appflow.dev.ibmappdomain.cloud/libertymq/api/enqueue

See the output in application runtime log from EAse Java UI:

[SystemO] Message enqueued.

[SystemO] MDB received: testing messages with IBM MQ on Cloud
